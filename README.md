# Coast FIRE Calculator

Angular web app for **Coast FIRE backtesting**: given a target portfolio (your FIRE number), current savings, and optional annual contributions or withdrawals, it runs simulations along **historic US equity return paths** to explore how many years it might take to reach the goal.

This is **educational software**, not financial advice. See the in-app disclaimer.

Mostly written by AI <3

## Development

From the `webapp` directory:

```bash
npm install
npm start
```

Then open `http://localhost:4200/`. For production build, tests, and linting, see `webapp/package.json` scripts.
